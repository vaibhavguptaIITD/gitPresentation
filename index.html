Simple git work flow

  1. Initialize a git repository: git init
  2. Configure git: git config
  3. Create a .gitIgnore file
  4. Add files in staging area: git add
  5. Commit files: git commit
  6. Check the logs: git log

Branching

Learning Git Branching
  • Create a branch : git branch <branch name>
  • Switch branches : git checkout <branch name>
  • Merge branches:
    • git merge <branch name>
    • git rebase <rebase onto branch name>
  • Resolving conflicts

Distributed Git workflow

  1. Clone a repository: git clone
  2. Push changes to repository: git push
  3. Get latest code
    • git fetch
    • git pull
  4. git remote
